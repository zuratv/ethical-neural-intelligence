# Ethical Neural Intelligence (ENI)

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.20352297.svg)](https://doi.org/10.5281/zenodo.20352297)

Public manifesto and reference home for Ethical Neural Intelligence.

Ethical Neural Intelligence (ENI) is a public philosophy and design direction for AI systems that remember, reason, explain, and remain accountable to human validation.

- Flagship: https://ethicalneural.ai
- First commercial surface: https://aiboard.live
- Research core: EniMind
- Manifesto: see [MANIFESTO.md](./MANIFESTO.md)
